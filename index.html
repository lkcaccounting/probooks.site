Connect Firebase

One-time setup — done by System Admin only

ℹ️ ProBooks needs a Firebase project to sync data across devices in real time. This is free and takes 5 minutes. Follow the instructions below, then paste your config here.
Steps:
1. Go to console.firebase.google.com
2. Click Add project → name it "ProBooks"
3. Click the </> Web icon to register app
4. Copy the firebaseConfig values below
5. In Build → Realtime Database, create a database
6. In Rules, set read/write to true (for now)
