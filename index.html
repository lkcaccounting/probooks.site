Install ProBooks on Your Device

Install ProBooks as a native app on any device — works offline once installed.

🤖 Android (Chrome)
  1. Tap the 3-dot menu (top right)
  2. Tap "Add to Home screen" or "Install app"
  3. Tap Add — done
🍎 iPhone / iPad (Safari)
  1. Tap the Share button (box with arrow at bottom)
  2. Scroll and tap "Add to Home Screen"
  3. Tap Add — done
💻 Windows / Mac / Linux (Chrome or Edge)
  1. Click the install icon in the address bar (looks like a screen with down arrow)
  2. Or: 3-dot menu → "Install ProBooks Accounting"
  3. Click Install — it opens like a native app

Once installed, ProBooks works completely offline. Internet is only needed for syncing data between devices and downloading PDF reports.
